# DeskRPG 0.1.3

Released: 2026-04-01

## Highlights

- Fixed `deskrpg init` for SQLite so it uses the runtime initializer instead of failing on duplicate Drizzle index creation.
- Added `deskrpg remove` to delete the local `.deskrpg` runtime home.
- Added `deskrpg uninstall` to remove the local runtime home and print uninstall guidance.

## Verification

- `npm run build`
- `DESKRPG_HOME=/tmp/deskrpg-cli-verify node bin/deskrpg.js init`
- `DESKRPG_HOME=/tmp/deskrpg-cli-verify node bin/deskrpg.js remove`
- Docker image push for `dandacompany/deskrpg:0.1.3`
