# DeskRPG 0.1.2

Released: 2026-04-01

## Highlights

- Fixed `deskrpg start` so the packaged CLI launches the custom DeskRPG runtime server instead of the standalone Next.js server.
- Restored packaged real-time connectivity by pointing production Socket.IO clients to the runtime socket port.
- Fixed AI connection state propagation so gateway setup updates immediately after saving channel settings.
- Added missing `npc.gatewaySetupHint` translations.
- Fixed packaged task automation runtime imports in `server.js` where `getTaskAutomationConfig` could resolve incorrectly.

## Verification

- `npm run build`
- `npm pack --dry-run`
- Docker image build and push for `dandacompany/deskrpg:0.1.2`
