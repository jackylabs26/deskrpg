# DeskRPG 0.1.6

Released: 2026-04-02

## Highlights

- Added user-owned OpenClaw gateway resources with explicit sharing.
- Added a dedicated `/gateways` management page for registering, testing, updating, and sharing gateways.
- Updated channel creation and channel settings so channels bind to one saved gateway or direct credentials.
- Channel gateway changes now warn before removing existing NPCs and related task or meeting context.
- NPC visibility now follows current gateway validity instead of showing stale DB rows when the bound gateway is unavailable.

## Notes

- This release changes the gateway model from inline channel credentials to reusable gateway resources with channel bindings.
- Existing production environments should validate channel-to-gateway bindings after upgrade if they previously relied on older inline gateway config behavior.
