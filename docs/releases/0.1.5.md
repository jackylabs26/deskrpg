# DeskRPG 0.1.5

Released: 2026-04-01

## Highlights

- Fixed fresh SQLite startup for both CLI installs and Docker SQLite hosting.
- Added runtime bootstrap for base SQLite tables so a brand-new database now creates core tables before compatibility backfills run.
- Included the new SQLite bootstrap helper in the production Docker image so container starts no longer fail with `no such table: users` or `no such table: channels`.
